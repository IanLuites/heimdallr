# Heimdallr

Hot code reload with code analysis application.


## Installation

The package can be installed
by adding `heimdallr` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:heimdallr, "~> 0.0.1"}
  ]
end
```

The docs can
be found at [https://hexdocs.pm/heimdallr](https://hexdocs.pm/heimdallr).
